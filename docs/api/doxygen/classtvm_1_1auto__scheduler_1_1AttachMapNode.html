Detailed Description

stores the compute_at relation between stages This stores a bi-directional mapping from stages and iter:

  1. Stage to its attached iterator
  2. Iterator to the stage attached to it You can use AttachMapNode::stage_to_attach_iter and AttachMapNode::iter_to_attached_stages to query the relations
