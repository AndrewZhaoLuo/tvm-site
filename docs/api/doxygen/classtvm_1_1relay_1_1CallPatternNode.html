tvm::Array<Type> tvm::relay::CallPatternNode::type_args

The type arguments passed to polymorphic(template) function.

This is the advance feature that is only used when the function is polymorphic. It is safe to be ignored in most cases. For example, in the following code, the type_args of addone call is [int].

template<typename T>
T addone(T a) { return a + 1; }
void main() {
int x = addone<int>(10);
}
